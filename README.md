# Online Image Editor | Debasis Prusty

🎨 **An online image editor built with HTML, CSS, and JavaScript**.  
You can upload images, apply filters, use presets, and download your creations instantly. Perfect for experimenting with colors! 🌈

## **Features**
- Upload images from your device  
- Apply multiple filters (brightness, contrast, saturation, etc.)  
- Use preset filters for quick effects  
- Download the edited image instantly  
- Fully responsive — works on **desktop and mobile**  

---

## **Technologies Used**
- **HTML5** – Structure of the editor  
- **CSS3** – Styling and responsiveness  
- **JavaScript** – Functionality, filters, presets, and canvas manipulation
